# Lab M6.01 - Deploy Application with Logging Configuration

## Overview
This lab demonstrates how to configure structured logging in a Flask application and send logs to AWS CloudWatch Logs using the CloudWatch Agent.

## Setup & Deployment
1. Created an EC2 instance
2. Deployed a Flask application with structured JSON logging
3. Installed and configured CloudWatch Agent
4. Created IAM role and policy for log permissions
5. Configured log shipping to CloudWatch

## Verification
- Logs generated locally in application.log
- Logs visible in CloudWatch Logs
- Logs queryable using Logs Insights
